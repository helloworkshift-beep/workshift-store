# AI Prompts for Contractors and Handymen: Win More Jobs, Get Paid Faster

The biggest challenge for most contractors isn't the work itself — it's the paperwork. Estimates, proposals, change orders, client follow-ups. These prompts handle the writing so you can stay on the tools.

## Project Proposal Prompts

### Full Project Proposal

```
Write a professional project proposal for a [PROJECT TYPE] job.
Client: [NAME]. Location: [CITY/STATE]. 
Scope of work: [DESCRIBE THE JOB IN DETAIL].
Timeline: [ESTIMATED DURATION]. Total cost: [$AMOUNT].
Include: brief company intro, detailed scope, what's included, what's NOT included, 
payment terms ([DEPOSIT]% upfront, balance on completion), and a simple call to action.
Tone: professional but approachable. Not a wall of legal text.
```

### Quick Quote Email

```
Write a short quote email for a [TYPE OF JOB] for [CLIENT NAME].
Scope: [BRIEF DESCRIPTION]. Total: [$AMOUNT]. Timeline: [START DATE / DURATION].
Keep it under 150 words. Professional, clear, easy to say yes to.
Include a simple next step (e.g., "Reply to confirm and I'll send the deposit invoice").
```

## Scope of Work and Change Orders

### Change Order Request

```
Write a change order request message for [CLIENT NAME].
Project: [PROJECT]. Original scope: [ORIGINAL SCOPE].
Newly discovered issue: [WHAT WAS FOUND]. Additional cost: [$AMOUNT]. Additional time: [DAYS].
Tone: matter-of-fact. Explain why it's necessary, what it involves, ask for approval.
Not apologetic — this is normal in construction.
```

## Client Communication Prompts

### Project Progress Update

```
Write a project progress update for [CLIENT NAME].
Project: [PROJECT]. What's been completed: [LIST].
What's happening this week: [PLAN].
Any issues or decisions needed from client: [ISSUES IF ANY].
Tone: professional, reassuring. Under 200 words.
```

### Late Payment Reminder (First)

```
Write a polite first payment reminder to [CLIENT NAME].
Invoice #[NUMBER] for [$AMOUNT] was due on [DATE] and is now [X] days overdue.
Tone: friendly but clear — this is a business, not a favor.
Include: invoice number, amount, original due date, how to pay. Under 100 words.
```

## Review and Referral Prompts

### Post-Project Review Request (Text)

```
Write a short text message to send after completing [PROJECT TYPE] for [CLIENT NAME].
Thank them for the job, briefly mention what was done, and ask for a Google review.
Include a placeholder for the review link. Natural tone. Under 60 words.
```
