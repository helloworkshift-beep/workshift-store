# AI Prompts for Teachers: Lesson Plans, Parent Emails, and Reports in Minutes

Teachers spend 10-15 hours per week on administrative tasks outside the classroom. Lesson planning, parent communication, report card comments, assessment creation. AI can cut that significantly.

## Lesson Planning Prompts

### Full Lesson Plan

```
Create a [X]-minute lesson plan for [SUBJECT], Grade [X], on the topic of [TOPIC].
Learning objective: students will be able to [OBJECTIVE].
Include: warm-up (5 min), direct instruction (15 min), guided practice (15 min), 
independent activity (10 min), exit ticket.
Note differentiation strategies for: early finishers, struggling students, ELL students.
Standards alignment: [STANDARDS IF KNOWN, OR SKIP].
```

### Discussion Questions

```
Generate 8 discussion questions for a [GRADE] class studying [TOPIC/TEXT].
Include: 2 recall questions, 3 analysis questions, 2 evaluation questions, 1 creative/personal connection question.
Use Bloom's Taxonomy levels. Questions should build on each other.
```

## Parent Communication Prompts

### Progress Update Email

```
Write a progress update email to the parent(s) of [STUDENT NAME], a [GRADE] student.
Current performance: [BRIEF ASSESSMENT].
Strengths: [2-3 SPECIFIC STRENGTHS].
Areas for growth: [1-2 AREAS].
Specific example of recent work or behavior: [EXAMPLE].
Tone: warm, honest, collaborative — not clinical.
Suggested next step for home: [SUGGESTION].
Under 200 words.
```

### Difficult Conversation Email (Behavior)

```
Write an email to the parent(s) of [STUDENT NAME] regarding [BEHAVIOR ISSUE].
What happened: [FACTUAL DESCRIPTION].
Impact: [ON CLASS / ON STUDENT].
What I've tried: [STEPS TAKEN].
What I'm asking for: [PARENT SUPPORT NEEDED].
Tone: collaborative, not accusatory. We're on the same team.
Under 250 words.
```

## Assessment Prompts

### Quiz Generator

```
Create a [X]-question quiz on [TOPIC] for Grade [X] students.
Question types: [MULTIPLE CHOICE / SHORT ANSWER / TRUE-FALSE — specify mix].
Difficulty: [BEGINNER / MIXED / CHALLENGING].
Include an answer key.
Align to: [LEARNING OBJECTIVE OR STANDARD].
```

### Report Card Comment Bank

```
Write 5 report card comments for a student who [PERFORMANCE DESCRIPTION].
Subject: [SUBJECT]. Grade level: [GRADE].
Vary the language across comments. 
Tone: specific, positive where possible, honest about growth areas.
Each comment: 2-3 sentences, under 60 words.
```

## The Real Value of AI for Teachers

The goal isn't to have AI teach your students — it's to give you more time with them. Every hour saved on lesson admin is an hour you can spend on the relationships and interventions that only a human teacher can provide.
