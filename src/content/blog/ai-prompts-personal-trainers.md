# AI Prompts for Personal Trainers: Grow Your Client Base and Deliver Better Programs

Personal trainers spend too much time on admin — writing programs, check-in messages, social posts — and not enough time actually training clients. These prompts fix that.

## Training Program Prompts

### Full Program Outline

```
Create a [X]-week training program for a [BEGINNER/INTERMEDIATE/ADVANCED] client.
Goal: [GOAL — fat loss / muscle gain / athletic performance / general fitness].
Equipment available: [GYM / HOME / MINIMAL / BODYWEIGHT ONLY].
Training days per week: [X]. Any limitations or injuries: [LIST OR NONE].
Include: phase breakdown, weekly session structure, sets/reps scheme, and progression notes.
Keep language motivating, not clinical.
```

### Single Session Plan

```
Write a [X]-minute workout session for a [GOAL]-focused client.
Equipment: [LIST]. Fitness level: [LEVEL]. 
Include: warm-up (5-10 min), main workout with sets/reps/rest, cool-down.
Add coaching cues for the 2 most technical exercises.
```

## Client Communication Prompts

### Weekly Check-In Message

```
Write a weekly check-in message to [CLIENT NAME].
They're working toward [GOAL]. This week they [DID WELL / STRUGGLED WITH]: [DETAIL].
Tone: encouraging, coach-like, specific. Not generic "great job!"
Ask one focused question about their week. Under 100 words.
```

### Plateau Response Message

```
Write a message to [CLIENT NAME] who has hit a [WEIGHT LOSS / STRENGTH / PERFORMANCE] plateau.
They've been consistent for [X] weeks. Two adjustments I'm making: [ADJUSTMENT 1] and [ADJUSTMENT 2].
Explain why plateaus happen, normalize it, introduce the changes with confidence.
Empathetic and energizing. Under 150 words.
```

## Sales and Onboarding Prompts

### Discovery Call Follow-Up

```
Write a follow-up message to [PROSPECT NAME] after a discovery call.
Their goal: [GOAL]. Their main concern: [CONCERN — usually price or time commitment].
Package I'm recommending: [PACKAGE] at [$PRICE/MONTH or per session].
Acknowledge their concern, connect the investment to their goal, include a clear next step.
Under 200 words.
```

## Social Media Prompts

### Transformation Story Post

```
Write an Instagram caption for a client transformation story.
Client: [FIRST NAME ONLY OR INITIALS]. Starting point: [BRIEF CONTEXT].
Result after [X weeks/months]: [RESULTS].
What made the difference: [KEY FACTOR].
Tone: inspiring but honest — no "results not typical" vibes. 
Include 5 relevant hashtags.
```

## Why Personal Trainers Need a Content System

Most trainers are great at training and terrible at marketing. Not because they don't have good things to say — but because writing takes time and creative energy that's already used up by actual coaching.

Prompts don't make you a writer. They make the writing faster, so you can be consistent without burning out.
